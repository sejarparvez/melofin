# Home Feed Personalization Plan

## Overview

Two-pronged approach: (1) fetch richer personalized content from InnerTube using additional browse endpoints, (2) track local play history to enable "Recently Played" / "Top Tracks" sections and feed deduplication.

### 1b. Add `browse_section()` in `innertube.rs`

Fetches a single browse_id and returns parsed sections (initial + one continuation):

```rust
pub(crate) fn browse_section(
    cookie_header: &str,
    api_key: &str,
    browse_id: &str,
) -> Result<Vec<HomeSection>> {
    let initial = browse_request(cookie_header, api_key, Some(browse_id), None)?;
    let mut sections = parse_sections(&initial);
    if let Some(token) = extract_continuation(&initial) {
        if let Ok(cont) = browse_request(cookie_header, api_key, None, Some(&token)) {
            sections.extend(parse_continuation(&cont));
        }
    }
    Ok(sections)
}
```

## Part 2: Local Play History

### 2a. New file: `src/play_history.rs`

**Data structure:**

```rust
#[derive(Clone, Debug, Serialize, Deserialize)]
pub struct PlayEvent {
    pub video_id: String,
    pub title: String,
    pub artist: String,
    pub thumbnail_url: String,
    pub timestamp: u64,       // unix seconds
    pub duration_played: f64, // seconds listened (0 = unknown)
    pub duration_total: f64,  // track length in seconds
}
```

**Helper to extract video_id from URL:**

```rust
pub fn video_id_from_url(url: &str) -> Option<String> {
    url.split_once("v=").map(|(_, rest)| rest.split('&').next().unwrap_or(rest).to_string())
}
```

**Storage:** Append-only JSONL at `~/.local/share/melofin/play_history.jsonl`. Capped at 500 entries (oldest trimmed when exceeded).

**Functions:**
- `load_history(path) -> Vec<PlayEvent>` — read all lines, deserialize
- `append_event(path, event)` — append one JSON line
- `trim_to_cap(path)` — keep newest 500, rewrite file (called after append when count > 500)
- `recently_played(events, limit) -> Vec<Track>` — deduplicate by video_id, take most recent `limit`
- `top_tracks(events, limit) -> Vec<Track>` — count plays per video_id, take top `limit`

### 2b. Hook into `PlayerCommand::Play` in `player.rs`

At line 99, after loading the track into mpv, record a play event:

```rust
PlayerCommand::Play(track) => {
    // ... existing mpv load + state send ...

    // Record play event (best-effort, non-blocking).
    let history_path = glib::user_data_dir()
        .join("melofin")
        .join("play_history.jsonl");
    let event = PlayEvent {
        video_id: video_id_from_url(&track.url).unwrap_or_default(),
        title: track.title.clone(),
        artist: track.artist.clone(),
        thumbnail_url: track.thumbnail_url.clone(),
        timestamp: std::time::SystemTime::now()
            .duration_since(std::time::UNIX_EPOCH)
            .unwrap_or_default()
            .as_secs(),
        duration_played: 0.0,
        duration_total: 0.0,
    };
    let _ = std::fs::create_dir_all(history_path.parent().unwrap());
    crate::play_history::append_event(&history_path, &event);
}
```

Note: `duration_played` and `duration_total` are set to 0.0 for v1 — the key signal is "which tracks were played" and "how many times". Accurate duration tracking can be added later by recording on track change.

## Part 3: UI Integration

### 3a. Merge local sections into home feed in `home_feed.rs`

After fetching the InnerTube feed (or yt-dlp fallback), load play history and prepend local sections:

```rust
fn merge_local_sections(feed: &mut HomeFeed, history_path: &Path) {
    let events = play_history::load_history(history_path);
    if events.is_empty() { return; }

    let recently = play_history::recently_played(&events, 20);
    if !recently.is_empty() {
        feed.sections.insert(0, HomeSection {
            title: "Recently Played".into(),
            tracks: recently,
        });
    }

    let top = play_history::top_tracks(&events, 20);
    if !top.is_empty() {
        let pos = if !recently.is_empty() { 1 } else { 0 };
        feed.sections.insert(pos, HomeSection {
            title: "Top Tracks".into(),
            tracks: top,
        });
    }
}
```

### 3b. Deduplication

Remove InnerTube tracks whose video_id appears in local sections:

```rust
fn dedup_video_ids(local_ids: &HashSet<String>, sections: &mut Vec<HomeSection>) {
    for section in sections {
        section.tracks.retain(|t| {
            play_history::video_id_from_url(&t.url)
                .map_or(true, |id| !local_ids.contains(&id))
        });
    }
    sections.retain(|s| !s.tracks.is_empty());
}
```
